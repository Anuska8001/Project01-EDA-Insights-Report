# Exploratory Data Analysis & Insights Report

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The objective is to identify patterns, trends, and insights through data cleaning, analysis, and visualization.

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Tasks
- Data Loading and Inspection
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Business Insights Report

## Key Findings
- Movies dominate Netflix's content library.
- The United States contributes the highest number of titles.
- Most content is rated TV-MA.
- Netflix experienced rapid growth after 2015.
